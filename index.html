Truth Table Generator

How to Use

Enter a logical statement in plain English. The generator will identify key concepts, map them to logical variables (P, Q, R), and create a complete truth table showing all possible outcomes.

Example Statements:

  • "If it rains then the grass gets wet"
  • "The car starts or the battery is dead but not both"
  • "Studying is equivalent to passing the exam"
  • "Working hard implies success"
  • "If Alice is present and Bob is absent, then Charlie is happy"

Supported Logical Operators:

  • AND: "and", "both"
  • OR: "or", "either", "alternatively"
  • NOT: "not", "isn't", "won't", "don't", etc.
  • XOR: "exclusive or", "either but not both", "mutually exclusive"
  • IMPLIES: "if...then", "implies", "leads to", "results in", "causes"
  • IFF (if and only if): "if and only if", "equivalent", "iff", "is equivalent to"

About This Tool

This Truth Table Generator uses natural language processing to convert English sentences into logical expressions. It can handle up to three distinct concepts (mapped to P, Q, and R) and generates complete truth tables for all possible combinations.

Perfect for students learning propositional logic, this tool helps visualize how different logical operators work together to determine the truth value of complex statements.
